## Development

1. Run the following command to the app infrastructure (ex. database) locally:

```

```

- C# Dev Kit by Microsoft
- C# by Microsoft

3. Install the Microsoft .NET SDK 9.0.306

4. Potentially some other steps that Robert did on my computer that I did not see/understand
